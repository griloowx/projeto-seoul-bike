# Previsão de Aluguel de Bicicletas em Seoul

Este projeto foi desenvolvido como parte do processo seletivo para estágio em uma consultoria de TI.

**Objetivo:** Prever a quantidade de bicicletas alugadas em Seoul com base em variáveis como temperatura, umidade, velocidade do vento e dia da semana.

## Arquivos

- `projeto_seoul_bike.ipynb` — notebook Jupyter com todo o código e análise
- `projeto_seoul_bike.html` — versão HTML gerada do notebook para visualização

## Tecnologias

- Python 3.x
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn
